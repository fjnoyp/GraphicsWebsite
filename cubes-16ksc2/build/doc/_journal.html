2014-09-06

Finished the java file. Tried to create an object oriented system for creating the .any files. I represent individual .any objects using java objects which have methods that print out string input. To aid in reusability and flexibility, only one java class handles all possible .any objects that can be created. Therefore the java class had methods that allowed for rotatin the object, setting its material, etc. A main class then converts those strings into the actual .any file by converting the java object instances into strings and then writing that to a separate .any file. Finished the staircase. Worked on the final custom scene. I used trig functions to position the objects and rotate them. I originally started with a tower of cubes that was rotated by a trig function. Afterwards I started changing cube scale with trig functions and then cube rotation as well. I then used orbit calls and other animation techniques to get the spiral movement that one can see in the movie. This entire process took about the entire day. Afterwards I made my movie by using the built in scene editor.
